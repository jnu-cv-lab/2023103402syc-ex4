# 计算机视觉实验四：采样、混叠与自适应下采样

## 实验环境
- 操作系统：Ubuntu / WSL2
- 开发工具：VS Code
- 编程语言：Python 3.12
- 依赖库：VS Code + Ubuntu (WSL) + OpenCV+NumPy+Matplotlib+SciPy

## 实验目的
1. 验证混叠失真与抗混叠处理
   - 实现直接下采样（`img[::M, ::M]`）与高斯平滑后下采样（`cv2.GaussianBlur` + 采样）
   - 对比两种方式在棋盘格和Chirp信号上的差异，理解预滤波对频谱混叠的抑制作用

2. 分析滤波参数 σ 对采样质量的影响
   - 验证理论最优值 σ = 0.45 × M（M为采样因子）
   - 对比 σ = 0.5, 1.0, 2.0, 4.0 时的频谱特性，确定最佳参数

3. 实现基于梯度的自适应下采样
   - 使用 Sobel 算子计算局部梯度，将图像分为 16×16 块
   - 根据梯度值动态调整采样率：高梯度区 M=2（保细节），低梯度区 M=4（高压缩）
   - 对比自适应策略与统一下采样（M=4）的 MSE 误差与视觉效果

4. 频域分析采样过程
   - 通过 FFT（`np.fft.fft2` + `fftshift`）观察下采样前后的频谱变化
   - 量化分析高频信息丢失与混叠现象的空间分布

5. 建立空间域与频域的关联
   - 结合空间域误差热力图与频域幅度谱，全面评价不同采样策略的保真度

## 实验原理与实现步骤

### 1. 测试图像生成

- 棋盘格图像：生成 256×256 棋盘格，块大小为 8 像素，用于验证高频混叠现象
- Chirp 图像：生成水平方向频率从 0 到 π 线性增加的正弦波，用于观察频率渐变的采样效果

### 2. 图像下采样

实现了两种下采样方式：

| 方法 | 原理 | 特点 |
|------|------|------|
| 直接下采样 | 每隔 M 个像素取一个 | 保留原始高频细节，但易产生锯齿和混叠失真 |
| 高斯滤波后下采样 | 先进行高斯模糊（σ=0.45×M），再采样 | 滤除高频噪声，有效抑制混叠，图像更平滑 |

最优 σ 理论：根据采样定理，最优高斯滤波参数为 σ = 0.45 × M（M 为采样因子）

### 3. 自适应下采样

基于图像局部梯度动态调整采样策略：

1. 梯度计算：使用 Sobel 算子计算图像梯度幅值
2. 分块处理：将图像分为 16×16 的块
3. 动态 M 值：
   - 高梯度区域（边缘）：M=2，σ=0.9（保留细节）
   - 中等梯度区域：M=3，σ=1.35
   - 低梯度区域（平滑）：M=4，σ=1.8（压缩率更高）
4. 重建统一：使用双三次插值将各块恢复到统一尺寸

### 4. 频域分析（FFT）

- 计算图像的二维傅里叶变换，并进行频谱中心化
- 幅度谱计算公式：`20 × log(|F(u,v)| + 1)`
- 观察指标：
  - 直接下采样：频谱出现高频泄漏（混叠）
  - 高斯滤波后：高频成分被有效抑制，频谱更干净

### 5. 空间域质量评价

- MSE（均方误差）：`MSE = mean((I_original - I_restored)²)`
- PSNR（峰值信噪比）：`PSNR = 10 × log10(255² / MSE)`

## 实验结果
Part 1：混叠现象验证
棋盘格结果：直接下采样出现明显的摩尔纹（混叠），高斯滤波后图像平滑
Chirp结果：高频区域（右侧）在直接采样中出现频率折叠，滤波后无此现象
频谱对比：直接采样频谱出现高频能量泄漏到低频区域
Part 2：σ 参数验证（M=4）
测试了 σ = 0.5, 1.0, 2.0, 4.0 四种参数：
σ = 0.5：滤波不足，仍有轻微混叠
σ = 1.8（理论最优）：最佳平衡，抗混叠效果与细节保留兼顾
σ = 4.0：过度平滑，丢失过多细节
Part 3：自适应下采样对比
与统一下采样（M=4）相比：
MSE 降低：自适应方法在边缘区域误差显著减小
误差热力图：自适应采样在边缘处误差分布更均匀
视觉质量：平滑区域压缩率高，边缘区域保持清晰
